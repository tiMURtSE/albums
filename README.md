# BORINGAHHMUSIC

## Описание?

Позволяет удобно коллекционировать альбомы.

## Функционал
+ Поиск альбомов
    + История поиска
+ Просмотр информации о найденных альбомов
+ Сохранение альбомов
+ Структурирование альбомов
    + Использование списков
    + Сортировка и фильтр списка
    + Добавление тегов (как на *Shikimori*)
    + Статистика
    + Оценки
    + Ссылки

## Страницы
+ Главная страница
    + Поиск
    + Красивый топ
    + Статистика
+ Список
+ Страница альбома
    + Обложка
    + Информация
    + Статистика
    + Комментарий
+ Настройки
